## 기능

- **헤더**
    - 컴포넌트 분리(기본 UI, 햄버거 메뉴)
    - Input checkbox 사용하여 햄버거 메뉴구현
- **견적요청**
    - 필수 정보 선택 및 입력 후 견적작성 api 기능구현
    - React-calendar 라이브러리 사용 및 css 수정
    - 모달 구현
- **위치기반**
    - 컴포넌트분리 (지역 메뉴바, 리스트 출력)
    - React-swiper 라이브러리 사용 및 css 수정
- **마이페이지**
    - 마이페이지 css
    - 현재 유저의 채팅 방 목록 출력
    - 클릭 한 채팅 방 연결
- **아이디찾기, 비밀번호찾기**
    - 비밀번호찾기 인증번호 입력
        - Input 입력 시 다음 Input으로 자동 탭 전환
    - 비밀번호 수정페이지 css
    - 아이디찾기 페이지 css
- **구슬님 상세정보 페이지**
    - 선택 한 구슬 상세정보 출력
    - 상담하기 클릭 시 해당 구슬님과 채팅방 생성
- **채팅**
    - 생성된 룸 ID로 채팅방 연결
    - 채팅방 입장 시 이전 대화목록 출력
    - 위로 스크롤 시 이전 대화내용 출력
